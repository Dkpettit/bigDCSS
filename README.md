# bigDCSS
A lightweight CSS Library written in SASS

<!-- start:code block -->
## Clone this repository
git clone https://github.com/Dkpettit/bigDCSS.git
cd bigDCSS

## Install dependencies
npm install

## Dependencies
+ Gulp
+ SASS
+ Gulp-SASS
+ Node

    npm init
    npm install gulp gulp-sass sass --save-dev
